如何搭建个人博客（一）

​ 搭建博客其实并不是想象中那样难以完成，现在有很多框架都可以协助大家简易的去构建一个属于自己的博客网站，例如本网站所使用的Hexo、Typecho以及它的主题网站等等，由以上两个框架，就可以建设出各式各样的个人网站。

​ 以下，我将以Hexo搭建博客为例，手把手教会大家如何搭建个人博客，就算你非计算机专业，无技术基础，通过以下步骤也可以成功的搭建出个人博客

一、搭建环境

1.安装Node.js

Node官网首页有如图两种选择，我们选择第一个推荐使用版，点击即可下载，下载后安装即可

安装完成后可以在cmd命令下执行 node -v命令，当出现了版本号，就表明安装成功了

node会自带npm

直接在cmd里执行npm -v，就可以验证是否有npm

2.安装Git

Windows: Git

安装完成后可以在cmd命令下执行 git -v命令，同样出现了版本号，就表明安装成功了

二、安装Hexo

1.设置npm为淘宝镜像源

为什么要设置npm为淘宝镜像源呢？

 因为node.js 默认使用的是国外的网站 。国内访问有一个跨国内局域网的操作。所以就会有时候很慢。这就跟为什么网站的静态资源有些会使用CDN 加速一样的
 淘宝镜像是什么？就是npm 很多的插件淘宝已经下载好了放在公共的网站上 我们需要的时候去淘宝网上下载 和 国外的是一样 这样使用是提升了我们的下载速度。

查看npm当前镜像源

设置npm为淘宝镜像源
